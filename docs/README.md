## Contributing to Documentation

When updating documentation:
1. Keep user docs separate from developer docs
2. Use clear headings and examples
3. Update this README if adding new docs
4. Keep the main README.md concise with links to detailed docs
5. Include code examples where helpful
6. Add screenshots for UI features (when available)
